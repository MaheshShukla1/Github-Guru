# Move into folder
cd github-guru

Open in VS Code or any editor and start exploring! ✨

🚀 Quick Start Example

Here's a simple git workflow to get you started:

  1. Initialize a repo: git init
  2. Add files: git add .
  3. Commit: git commit -m "First commit"
